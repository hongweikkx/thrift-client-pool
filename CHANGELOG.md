# Changelog

所有对此项目的重要更改都将记录在此文件中。

## [1.0.0] - 2026-05-09

### 新增
- 升级 Go 到 1.25，提供最新的语言特性和安全补丁
- 升级 Apache Thrift 到 v0.20.0，获得最新的功能和修复
- 创建 `pool_test.go`，包含 12 个单元测试用例

### 改进
- 完善了代码注释，所有导出函数都添加了文档
- 改进了错误处理，使用 `fmt.Errorf` 提供更清晰的错误信息
- 添加了 `ReturnObject` 方法到 Pool，简化 API 使用
- 代码格式化，符合 Go 标准

### 文档
- 重写了 README.md，包含详细的使用示例和 API 文档
- 添加了快速开始指南
- 添加了常见问题解答 (FAQ)
- 创建了 CONTRIBUTING.md 贡献指南

### 测试
- 添加了连接池核心功能的单元测试
- 测试包括：配置、借用/归还、并发处理、Context 取消等
- 所有测试无外部依赖，100% 本地运行

## 之前的版本

### 原始版本
- 实现基于对象池的 Thrift 连接池
- 支持多种协议和传输方式
- 演示 1000 个请求下 TIME_WAIT 优化效果
